# xyzlab.dev

Personal developer portfolio site for [chr0nzz](https://github.com/chr0nzz). Built with [Astro](https://astro.build) and hosted on Cloudflare Pages.

## Stack

- [Astro](https://astro.build) - static site generator
- GitHub API - live repo stats, recent commits, open issues
- Cloudflare Pages - hosting

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`. Cloudflare Pages runs the build automatically on push to `main`.
